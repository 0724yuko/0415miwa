# 顧客対応管理

## 概要
グループ内の担当者ごとの分担業務の管理と顧客情報の管理を容易にします。

顧客情報の管理。

顧客案件内容の管理、案件の日程管理。

担当者ごとの案件管理。

上記の顧客管理・案件管理・分担進捗管理を一元化します。

## 主な機能
- ログイン・ログアウト機能
- 顧客登録・修正・削除
- 顧客一覧表示
- 顧客検索
- 顧客対応案件登録・修正・削除
- 顧客ごとの対応案件一覧表示
- 担当者一覧表示
- 担当者ごとの対応案件一覧表示

## 開発環境

```
PHP 8.2.4
MySQL 15.1
Laravel 10.13.5
```
